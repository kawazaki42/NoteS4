# Введение. Основные термины и опреледения

__Предметная область__: часть материального мира, которая подлежит моделированию...

может расширятся, но не надо переусердствовать

- объекты
- процессы


__Данные__: некоторая информация в предметной области...

__База данных__: _цифровой слепок_ этой информации

__Информационные системы__: ПО, основной носитель баз данных

есть некоторые знания

__Операции с данными__: CRUD

- Create
- Read
- Update
- Delete

но они по разному назваются в разных языках

__Метаданные__: _данные о данных_; служебная информация

отличаются от просто данных

но нету четкой границы

- тип атрибутов
- время занесения строки (логи)

---


данные это то что внутри табличек

заголовки это метаданные

мету можно загнать в обычные данные!

и наоборот

__Модель данных__: термин зависит от контекста

1. структура данных
1. Интерфейс манипулирования
1. Ограничения целостности

- Реляционные модели
  - через таблицы
  - интерфейс - sql
  - ключи, проверки
- документоориентирование
  - json
- графовые
  - тоже нет жесткой схемы


__Модель данных__: некая абстракция....
1. Потоков данных
2. Сущность-связь

__СУБД__: ПО для работы с базами данных
1. Манипуляция данными на диске
2. Манипуляция дынными в памяти
3. Поддержка языка манипулирования
4. Поддержка целостности
   - встроенное резервное копирование
5. Дополнительные функции
   - распределение ролей
   - статистики
