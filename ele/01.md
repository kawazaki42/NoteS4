  - источника электрической энергии

## Режимы работы источников электрической энергии

В соответствии с подключаемой нагрузкой различают следующие режимы работы:
- Холостой ход (ХХ)
  - Возникает в случае размыкания электрической цепи
  - R_н = \infty
  - I_н = 0
  - U = E
- Короткое замыкание (КЗ)
  - R_н = 0
  - I_н = E / R_0
  - U = 0
- Номинальный (НОМ)
  - это режим работы нагрузки с заданными параметрами
  - R_н = R_ном
  - I_н = I_ном
  - U_н = U_ном
- Согласованный (СОГЛ)
  - возникает при R_0 = R_н
  - I_н = 0.5 I_кз
  - U_н = 0.5 E
  - характеризуется выделением нагрузки максимальной мощности
  - $\eta = 50%$

## Топографические понятия схемы замещения

1. Ветвь электрической цепи
   - это участок электрической цепи, по которому протекает
   - один и тот же ток
2. Узел электрической цепи
   - это точка электрической цепи, где соединяется 3 и более ветвей
3. Контур электрической цепи
   - это любой замкнутый путь по узлам и ветвям схемы

## Законы Кирхгофа

1. Алгебраическая сумма токов, сходящихся в узле равна нулю.
   - $\sum _{i=1} ^n I_i = 0$
   - Закон направления: "Ot baldy"
   - a) -I_1 +I_4 +I_6 = 0
   - b) +I_1 +I_2 -I_3 = 0
   - d) I_3 + I_5 - I_6 = 0
2. Алгебраическая сумма падений напряжений в любом контуре электрической цепи
   равна алгебраической суммы ЭДС этого же контура.
   - $\sum _{i=1} ^n I_i R_i = \sum _{j=1} ^m E_j$

По втотрому закону Кирхгофа составляется столько уравнений,
сколько есть независимых контуров.

Контур будем считать независимым,
если при его рассмотрении в него входит хотя бы одна
новая ветвь, не входящая в другие контуры.

1) + I_1 R_1 - I_2 R_2 - I_4 R_4 = + E_1 - E_2
2) - I_2 R_2 - I_3 R_3 + I_5 R_5 = - E_2 + E_3
3) + I_1 R_1 - I_2 R_2 + I_5 R_5 + I_6 R_6 = + E_1 - E_2

Для уменьшения количества уравнений используются расчетные методы:
- котнурных токов (МКТ)
  - основан на уравнениях по 2 закону Киргофа
  - и допущении, что в каждом независимом контуре
    - протекает свой независимый контурный ток
- узловых потенциалов (или напряжений) (МУП)
- эквивалентного генератора (МЭГ)
  - для одного тока...
- наложения
  - убираем по два источника...
- ...

- I_к1 (R_1 + R_2 + R_4) + I_к2 R_2 + I_к3 (R_1 + R_2) = E_1 - E-2
- I_к2 (R_2 + R_3 + R_5) + I_к1 R_2 + I_к3 (R_2 + R_5) = E_3 - E-2
- I_к3 (R_1 + R_2 + R_5 + R_6) + I_к1 (R_1 + R_2) + I_к2 (R_2 + R_5) = E_1 - E_2

## Баланс мощностей

Поскольку процесс выработки, передачи и потребления электрической энергии
неразрывно связаны, это позволяет нам сформулировать баланс мощности:

> Сколько мощности потребляется потребителем электрической энергии,
> столько в этот момент мощности вырабатывается источником
>
> P_ист = P_нагр

$$Q = I^2 R t$$

$$P_{нагр} = I^2 R$$

$$\rArr P_{нагр} = UI = U^2 / R$$

$$P_ИП = EI$$

Вот так можно перепровериться:

$$\sum _{i=1} ^n E_i I_i = \sum _{j=1} ^m I_j ^2 \cdot R_j$$

Для рассмотренной ранее схемы можно составить уравнение баланса мощности

$$P_ИП = + E_1 I_1 - E_2 I_2 - E_3 I_3$$

а вот тут отрицательно быть не должно

$$P_нагр = I_1 ^2 R_1  I_2 ^2 R_2  I_3 ^2 R_3  I_4 ^2 R_4  I_5 ^2 R_5  I_6 ^2 R_6$$
